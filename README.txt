EdgeDetection.py:

Run this file in order to obtain an image with the edges highlighted
Function definitions are in the beginning with the main block of code starting on line 67
The only parameters that should be changed are: sigma and the image file located on line 68
Line 89 can also be uncommented in order to save the output image developed

CCLabel.py:

Code will return the number of distinct objects based on an edge detection image provided
Simply run the EdgeDetection.py on any desired image and save the output image
Change line 30 of CCLabel.py to the output file desired in order to determine the total number of objects
